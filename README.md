# A-Java-program-for-Cell-phone-PIN-number-and-PUK-code-using-method-and-Control-Structures.

Create a Java program for Cell phone PIN number and PUK code using method and Control Structures.  If the user entered the correct pin code displays “Welcome to Sir Jay Telecom, A Telecom that Always On-Line”.  The cell phone user will enter his pin code three times; if on the first, second and third trial he failed to enter the correct pin code it will display “Invalid Pin Number”, after the third will prompt the user to enter PUK secret code three times also.  If the user still failed to enter PUK code will display “SIM BLOCKED and You Are Busted”.

Pin number: 0000
PUK Code: 12345678
